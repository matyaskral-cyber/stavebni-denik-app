Instalace agenta
1. Stáhněte soubory agenta
Zkopírujte složku agent/ na váš server.
2. Vytvořte config.json
{ "api_url": "{{ request.url_root }}{{ firma.slug }}/api/backup/export", "api_key": "VÁŠ_API_KLÍČ", "local_path": "/opt/backups/stavebni-denik", "firma_slug": "{{ firma.slug }}" }
3. Spusťte agenta
npm install node sync-agent.js
4. Nastavte cron (volitelné)
# Každý den v 4:00 0 4 * * * cd /opt/agent && node sync-agent.js >> /var/log/backup.log 2>&1
